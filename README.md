# 会捣乱的积木

一款轻松又烧脑的限时拼图小游戏。你是一名"积木工匠"，要在时间耗尽前，照着图纸把每个格子填上正确颜色，完成顾客的订单。可这些积木一个个都有自己的脾气——会烧邻居、会疯长、会腐蚀、会护盘。趁它们捣乱之前，又快又准地拼出图案，就是这场博弈的全部乐趣。

---

## 游戏目标

- 每一关都有一张**图纸**（虚线轮廓标出每格的目标颜色）。
- 在**倒计时归零前**，把图纸上所有目标格都填上对应颜色的积木。
- 填对的格子会亮起金边；全部填对即**完成订单**，进入下一关。
- 时间到还没完成，则本关失败，可重试。

---

## 基本操作

- **放置**：在左侧材料库选中一种积木，点击画布上的空格即可放下。
- **拆除/回收**：点击自己放下的积木可以拆掉，积木回到材料库重复使用。
- **暂停**：点顶栏"⏸ 暂停"按钮，或按**空格键**随时暂停 / 继续。
- **音效开关**：顶栏"🔊 音效 / 🔇 静音"一键切换，所有音效与背景音乐随之开关（选择会被记住）。
- **重置本关**：顶栏"↻ 重置本关"重新开始当前关卡。
- **关卡选择**：底部关卡条可切换已解锁的关卡；未通关的关卡处于锁定状态。

## 积木图鉴

每种积木都有独特的"副作用"，理解它们的脾气是通关的关键。

| 积木 | 性格 | 行为说明 |
|------|------|----------|
| 🟡 **黄砖** | 稳定 | 老实本分，没有任何副作用，是最可靠的填充材料。 |
| 🔴 **红砖** | 暴躁·烧裂 | 会持续灼烧**相邻**的积木，被烧的积木逐渐出现裂纹，裂满后碎裂消失（带碎屑与烟雾）。 |
| 🟢 **绿砖** | 疯长·蔓延 | 会不断向相邻的**空格**蔓延出新的绿块。被蔓延占据的格子会变暗、锁死，不能再拆改，只能用红砖烧掉。 |
| 🔵 **蓝砖** | 不稳定·腐蚀 | 放下后撑不了几秒就会自行腐蚀消失（冒泡溶解）。适合留到最后再填。 |
| ⚫ **黑砖** | 稳重·护盘 | 自身不会被烧，还能**保护相邻积木**不被红砖灼烧，是对抗火焰的盾牌。 |
| 🌈 **彩虹砖** | 万能 | 可以匹配任意颜色的目标格，数量稀少，是关键时刻的救命底牌。 |
| 🪨 **石头障碍** | 障碍物 | 不能点击、不能拆除。需用**相邻的红砖把它烧碎**，碎裂后露出待填格，再由你补上对应颜色。 |

> 小贴士：红砖怕黑砖（护盘免疫），绿砖怕红砖（会被烧掉），蓝砖自己会消失。学会用一种积木的脾气去克制另一种，是高手的玩法。

## 核心机制

- **灼烧是渐进的**：红砖不是瞬间烧掉邻居，而是从挨上的那一刻起逐渐积累裂纹，裂满才碎裂。被多块红砖夹住会烧得更快；红砖移走或被黑砖护住，裂纹会缓慢自愈。
- **黑砖护盘**：任何与黑砖相邻的积木都免疫灼烧。合理布置黑砖能护住关键格子。
- **绿砖蔓延**：绿砖按固定节奏向空格扩散。手慢就会被绿块吞掉地盘，且被占据的格子会锁死，得用红砖清理。
- **蓝砖倒计时**：蓝砖放下后会在几个结算回合内腐蚀消失，需要掐准时机最后再填。
- **石头障碍**：作为关卡里的"硬骨头"，必须用火（红砖）烧碎才能腾出格子。
- **策略性**：很多关卡考验**放置顺序**——比如先用黑砖搭好护盾再放火砖，或先清理蔓延的绿块再填核心格。

## 关卡介绍

游戏共 **7 关**，难度循序渐进，逐步引入新积木与新机制。

1. **第一关 · 填充** — 新手引导。只用黄砖，熟悉放置与填充的基本操作。
2. **第二关 · 毁灭** — 认识红砖的灼烧。角落还有石头障碍，要用红砖烧碎它们再补填。
3. **第三关 · 生长** — 认识绿砖的蔓延，体验它疯长的威力，小心地盘被永久侵占。
4. **第四关 · 协调** — 红砖与绿砖两个捣蛋鬼同台，时间紧迫，要善用它们各自的特性。
5. **第五关 · 距离** — 学会"惹不起就躲远点"，用布局上的距离化解暴躁的脾气。
6. **第六关 · 抵抗** — 再顽强的生命力也抵不过火焰，考验灼烧与蔓延的对抗。
7. **第七关 · 保护** — 彩蛋收尾关，集合多种积木与石头障碍的综合挑战，等你自行探索。

---

## 通关小技巧

- **先看图纸再动手**：规划好哪些格子用什么颜色，避免手忙脚乱。
- **顺序很重要**：会消失/会蔓延的积木尽量留到合适时机；需要保护的地方先放黑砖。
- **化敌为友**：红砖能帮你清掉碍事的绿块和石头；蓝砖会自动腾位。
- **盯紧倒计时**：剩余时间在 10 秒内会变红警示，合理分配最后的填充节奏。
- **善用彩虹砖**：数量有限，留给最难搞的格子。
